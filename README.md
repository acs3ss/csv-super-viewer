# CSV Super Viewer
Edit CSV files from the web

https://acs3ss.github.io/csv-super-viewer/

## Developing

1. `npm install`
2. `npm run webpack:watch`
3. `npm run css:watch`
4. Open https://localhost:3001 in your browser and accept any security warnings

## Publishing
1. `npm run build`
2. `cd gh-pages`
3. `git commit -a`
4. `git push`
